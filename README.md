# PERN Store — Azure CI/CD Lab

## Live URLs
- Frontend: https://purple-water-066a40600.7.azurestaticapps.net
- Backend API: https://app-pernstore-backend-iqra-hnahbvf2d0fzeqdh.eastasia-01.azurewebsites.net

## Architecture
- Frontend: React.js deployed to Azure Static Web App
- Backend: Node.js/Express (Docker) deployed to Azure App Service
- Database: PostgreSQL on Azure Database for PostgreSQL
- Registry: Docker Images on Azure Container Registry
- CI/CD: Azure DevOps YAML Pipelines

## Pipeline
Multi-stage pipeline: Build, Push to ACR, Deploy to Production
